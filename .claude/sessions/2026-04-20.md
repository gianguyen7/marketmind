# 2026-04-20

## Session 01:00 — Replace Streamlit with MarketMind design system UI + one-pager

### Context
User provided a complete MarketMind design system (from Downloads) with CSS tokens, UI kit components (React/JSX), preview specimens, logo assets, and a findings one-pager HTML. Goal was two deliverables: (1) replace the Streamlit dashboard with a static HTML/React dashboard wired to real data, and (2) integrate the findings one-pager.

### New information
- MarketMind design system exists as a standalone skill package with colors_and_type.css, preview cards, and a full React UI kit in `ui_kits/dashboard/`
- Design system uses Source Serif 4 (display), Inter (body), JetBrains Mono (numbers) via Google Fonts CDN
- Design tokens: `--mm-indigo` (#4F5BD5) primary accent, `--mm-paper` (#FAFAF7) warm off-white bg, 9-step warm neutral ramp
- UI kit implements 5 pages as React components with Babel standalone (no build step)
- One-pager has a built-in tweaks panel (accent color, paper tone, dense mode, section toggles)

### Decisions
- **Static HTML/React via CDN, no build tools** — user explicitly chose this over Vite/webpack. React 18.3.1 + Babel standalone transpiles JSX in-browser
- **Wire to existing CSV/JSON data outputs** — dashboard fetches from `../../outputs/tables/` relative paths. Requires HTTP server (serve.py provided)
- **Old Streamlit app preserved as app.py.bak** — not deleted, renamed for reference
- **Volume-by-year data hardcoded in DataPage** — source is parquet (not loadable in browser), all other data wired to CSVs/JSON
- **Remove all em/en dashes from copywriting** — user preference, replaced with periods, commas, parentheses throughout both dashboard and one-pager
- **Friendly model names with tooltips** — technical model names (isotonic_global, platt_per_cat, etc.) replaced with human-readable names + non-technical tooltip explanations
- **Remove B5/C1/C4 prefixes** — internal experiment codes removed from all user-facing labels

### Implementations
- `src/dashboard/index.html` — main entry point, loads React/Babel CDN + 5 JSX modules
- `src/dashboard/colors_and_type.css` — full design system tokens (copied from Downloads)
- `src/dashboard/styles.css` — component styles: sidebar, tiles, tables, callouts, tabs, charts
- `src/dashboard/Primitives.jsx` — MetricTile, Callout, Badge, Button, DataTable, TabBar, SectionHeader
- `src/dashboard/DataLoader.jsx` — CSV parser, fetch with caching, useData hook, category name mapping, number formatters, LoadingState/ErrorState components
- `src/dashboard/Charts.jsx` — ReliabilityChart, BiasBars, SplitBars, CategoryBar, VolumeOverTime (SVG)
- `src/dashboard/Sidebar.jsx` — 5-page navigation with Lucide-style SVG icons
- `src/dashboard/Pages.jsx` — 5 data-driven pages: OverviewPage (with research directions section), CalibrationPage, FLBiasPage, ExploitPage (4 tabs), DataPage. MODEL_INFO map with tooltips. All wired to CSV/JSON via fetchCSV/fetchJSON
- `src/dashboard/one-pager.html` — standalone findings one-pager with data-loading script that fetches all metrics, chart data, split table, exploitation deltas, key finding numbers from outputs/tables/
- `src/dashboard/serve.py` — HTTP server that serves from project root (port 8000 default, configurable)
- `src/dashboard/assets/logo-mark.svg, logo-full.svg, logo-mono.svg` — brand logos
